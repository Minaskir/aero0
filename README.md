# АэроКафе - Сайт сети кафе в аэропорту

## Описание
Современный веб-сайт для сети из трех кафе в аэропорту: Манго, Монте Карло и Какаду.

## История обновлений

### Обновления от 9 мая 2026 г.
Масштабное расширение функционала платформы:

#### 1. Обновление дизайна и логики меню
- **Новый UI:** Полностью переработана страница меню — добавлены анимированные карточки, поиск по названию/составу и удобные фильтры по категориям.
- **Вариативность блюд:** Реализована поддержка вариантов для напитков (выбор объема с автоматическим пересчетом цены).
- **Стоп-листы:** Добавлена возможность скрывать позиции из меню в один клик через админ-панель.

#### 2. Расширение админ-панели
- **Вкладка "Настройки":** Появился раздел для управления глобальными параметрами системы (ключи API, контактные данные, время ожидания заказа).
- **Массовые рассылки:** Интегрирован конструктор email-сообщений. Теперь администратор может отправлять красивые HTML-письма всей базе клиентов прямо из панели управления.
- **Управление пользователями:** Добавлен список клиентов с подробной статистикой (LTV, количество заказов) и возможностью блокировки или изменения прав доступа.
- **Логирование (Аудит):** Внедрена система записи всех важных действий администраторов (изменение цен, удаление позиций, рассылки) для повышения безопасности.

#### 3. Сервисные функции
- **HR-модуль:** Добавлена страница вакансий и функционал подачи анкет. В админ-панели появился раздел для обработки откликов и управления статусами кандидатов.
- **Личные уведомления:** Реализована отправка сообщений в личный кабинет пользователя (уведомления о готовности, бонусы и т.д.).
- **Восстановление доступа:** Запущена система сброса пароля через Email с использованием безопасных временных токенов.
- **Экспорт данных:** Добавлена выгрузка бронирований и анкет соискателей в формат Excel для бухгалтерии и отдела кадров.
- **Симуляция кухни:** На бэкенде запущена служба автоматического обновления статусов в заказах (Принят → Готовится → Готов → Выдан), имитирующая реальный процесс работы заведений.

### Обновления от 10 мая 2026 г.
Работы по запуску, наполнению и интеграции внешних сервисов:

#### 1. Глобальное обновление меню
- **Визуализация:** Ко всем блюдам добавлены качественные фотографии, что значительно улучшило визуальное восприятие меню.

- **Новые разделы:** В меню полностью интегрированы разделы с напитками, включая обширную кофейную и чайную карты, безалкогольные напитки и барное меню.

#### 2. Полноценный запуск проекта
- Установлены все необходимые зависимости через `npm install`.
- Настроен бэкенд на Node.js (Express) с подключением к БД PostgreSQL.
- Создана база данных и выполнена миграция таблиц.
- Реализован запуск сервера на порту 3000.

#### 3. Наполнение данными
- Выполнен финальный скрипт `seed-menu.js` для первичного наполнения системы.
- Создана учетная запись администратора (`admin@aerocafe.ru` / `admin`) для управления системой.

#### 4. Реальное табло рейсов (Оренбург - REN)
- **Бэкенд-прокси:** Добавлен API-маршрут `/api/flights/:event` для безопасного получения данных.
- **Интеграция с Яндекс Расписаниями:** Настроено получение живого расписания аэропорта Оренбург.

## Технологии
- **Frontend:** HTML5, CSS3, Vanilla JS
- **Backend:** Node.js, Express, Sequelize
- **Database:** PostgreSQL
- **API:** Яндекс Расписания (Online Board)

## Установка и запуск
1. Установите зависимости: `npm install`
2. Настройте `.env` (БД, JWT, YANDEX_API_KEY)
3. Заполните базу: `node seed-menu.js`
4. Запустите сервер: `npm start`
5. Откройте `http://localhost:3000`
